Project 8: Radio and WiFi

Guts of a Smart Garage Door

This week was a partner week! I teamed up with Camtastic and we were tasked with using our knowledge of WiFi and Arduino programming and coding to create a device with one input and one output that would operate over WiFi. Cam suggested creating something similar to a garage door opener with a button and a DC motor, and I thought it was a great idea!


The idea for the project is having a button on one breadboard while the other user stands across the room and once the button is pressed, a motor on their breadboard can spin via WiFi. Here are the code snippets we used for the project. We took inspiration from Rui's code on the PS70 website:
