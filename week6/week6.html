Project 6: Output Devices

Servo and Thermistor

This week, I was tasked with creating an input/output device as well as a minimum viable final project.


I decided that I could kill two birds with one stone and start working on my MVP with the input/output(for my final puzzle box project).

For the sensor, I set up a temperature sensor with a thermistor. I played with the serial plotter and got my sensor to work! The plot corresponds to the temperature that the thermistor reads, in degrees Celsius.

I added a servo motor to the thermistor so that when the thermistor reaches a threshold of 25 degrees Celsius, the servo turns.
